## Running Locally

1. Clone the repository

```bash
git clone <PROJECT_REPOSITORY>.git
```

2. Install dependencies using npm

```bash
npm install
```

3. Copy the `.env.example` to `.env` and update the variables.

```bash
cp .env.example .env
```

4. Start the development server

```bash
npm run dev
```
